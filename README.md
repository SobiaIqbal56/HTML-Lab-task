# HTML-Lab-task
This repository includes a set of HTML lab assignments designed to practice core concepts of web development.
## Description
- Task 1 focuses on different types of HTML lists including ordered lists, unordered lists, and definition lists with proper nesting.
- Task 2 implements a registration form using HTML form controls arranged in a table layout.
- Task 3 demonstrates HTML5 features such as color input, number restriction, date picker, audio element, and canvas for drawing text.
These tasks are created for learning and academic purposes.
